Detailed Description

template<class T1, class T2>
class fnss::Pair< T1, T2 >

Wrapper class for std::pair that adds optional commutativity to the pair, eg. the pairs <1, 2> and <2, 1> will return true for operator==(..) if the commutative flag is set.
